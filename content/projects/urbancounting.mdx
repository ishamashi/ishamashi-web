## Purpose

UrbanCounting adalah "wajah" komersial dari teknologi AdMeasurement. Jika AdMeasurement adalah hardware-nya, UrbanCounting adalah _brand_ layanannya.

## Features

- **Interactive Demo:** Menampilkan simulasi data trafik secara visual untuk meyakinkan calon klien.
- **Service Catalog:** Penjelasan detail tentang layanan survei trafik berbasis AI.

## Role

Sebagai Founder, saya membangun web ini untuk mendukung aktivitas _Sales & Marketing_, mengubah jargon teknis IoT menjadi proposisi nilai bisnis yang mudah dipahami klien.
