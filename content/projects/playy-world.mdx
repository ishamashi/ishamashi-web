## Context

Bekerja secara remote untuk **Playy.World Pte. Ltd (Singapore)**. Ini adalah salah satu pengalaman awal saya bekerja dengan standar internasional.

## Engineering Role

Sebagai **Fullstack Developer**, saya bertanggung jawab membangun fitur inti marketplace:

- **Listing Engine:** Memungkinkan user mengunggah kartu dengan atribut spesifik (Rarity, Condition, Set).
- **Transaction Flow:** Mengelola logic _Escrow_ sederhana untuk memastikan keamanan antara penjual dan pembeli.

## Value

Proyek ini mengasah kemampuan komunikasi lintas negara dan pemahaman tentang logika bisnis _e-commerce_ (Cart, Checkout, Order Management).
