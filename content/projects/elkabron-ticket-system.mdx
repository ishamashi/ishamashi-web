## Overview

Untuk event spesial (DJ Guest/Parties), ElKabron membutuhkan kontrol penuh atas penjualan tiket tanpa bergantung pada potongan komisi platform luar yang besar.

## Features

- **Direct Payment:** Integrasi payment gateway lokal dan internasional.
- **Rapid Validation:** Aplikasi scanner khusus untuk security di pintu masuk, mampu memvalidasi tiket < 1 detik per orang.
- **Reporting:** Laporan penjualan tiket real-time untuk tim Finance.

## Value

Menghilangkan ketergantungan pada vendor tiket eksternal dan memberikan data audiens (CRM) sepenuhnya kepada manajemen.
