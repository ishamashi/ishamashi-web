## The Problem

Inspeksi ribuan tabung pemadam api di gudang raksasa dilakukan secara manual dengan kartu gantung kertas. Data sering hilang, rusak, atau dipalsukan (tidak dicek tapi dicentang).

## The Innovation

Digitalisasi proses inspeksi K3 (Keselamatan & Kesehatan Kerja).

- **QR Scanning:** Petugas wajib scan QR code di tabung fisik untuk input data inspeksi (memastikan kehadiran fisik).
- **Expiration Alert:** Notifikasi otomatis jika ada tabung yang mendekati masa kadaluarsa (expired).

## Strategic Value

Memastikan kepatuhan (Compliance) terhadap standar keamanan industri dan ISO, serta memberikan visibilitas total kepada manajer gudang.
